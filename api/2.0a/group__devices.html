Detailed Description

A device is an entity on the distributed graph which has input and/or output signals. The mpr_dev is the primary interface through which a program uses libmapper. A device must have a name, to which a unique ordinal is subsequently appended. It can also be given other user-specified metadata. Device signals can be connected, which is accomplished by requests from an external GUI or session manager.
